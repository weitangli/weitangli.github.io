爬虫框架webmagic

https://webmagic.io

WebMagic是一个简单灵活的Java爬虫框架。基于WebMagic，你可以快速开发出一个高效、易维护的爬虫。

特性：

  • 简单的API，可快速上手
  • 模块化的结构，可轻松扩展
  • 提供多线程和分布式支持
