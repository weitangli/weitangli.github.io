ActiveMQ的简易安装

安装ActiveMQ之前, 我们得先安装Java运行环境.

Linux安装包下载地址: apache-activemq-5.14.5-bin.tar.gz

解压后的目录结构如下:

bin目录下根据不同的操作系统, 提供了不同的启动脚本, 包括: linux-x86-32, linux-x86-64, macosx.

我的服务器是centos7.1系统, 使用的是bin/linux-x86-64/activemq脚本.

  • 启动服务 bin/linux-x86-64/activemq start
  • 查看状态 bin/linux-x86-64/activemq status

管理后台: http://localhost:8161
默认账号/密码: admin/admin
