快速安装Redis

  • 官方网站 https://redis.io
  • 中文网站 http://www.redis.cn

Redis是一种高级的Key-Value存储系统, 其中Value支持五种数据类型:

  • string 字符串
  • list 列表
  • set 集合
  • sorted set 有序集合
  • hash 哈希
